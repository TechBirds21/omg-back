# O Maguva - Premium Sarees & Traditional Wear

## Project Description

O Maguva is an e-commerce platform specializing in premium sarees and traditional wear. This React-based application provides a complete shopping experience with modern features and traditional aesthetics.

## Technologies Used

This project is built with:

- **React** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and development server
- **Tailwind CSS** - Styling framework
- **shadcn/ui** - UI component library
- **Supabase** - Backend and database
- **PhonePe & Easebuzz** - Payment gateway integration

## Getting Started

### Prerequisites
- Node.js (18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repository-url>

# Navigate to project directory
cd bag-bash

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
# Build the application
npm run build

# Preview the production build
npm run preview
```

## Features

- **Product Catalog** - Browse premium sarees and traditional wear
- **Shopping Cart** - Add items and manage orders
- **User Authentication** - Secure login and registration
- **Payment Integration** - PhonePe and Easebuzz payment gateways
- **Order Tracking** - Track order status and delivery
- **Admin Panel** - Manage inventory, products, and orders
- **Responsive Design** - Mobile-friendly interface

## Payment Integration

The application supports multiple payment gateways:

### PhonePe Integration
- Secure payment initiation
- UPI, cards, and net banking support
- Payment verification and confirmation
- Invoice generation and email delivery

### Easebuzz Integration
- Alternative payment gateway
- Multiple payment methods
- Webhook support for payment status updates

## Admin Features

Admin panel includes:
- **Inventory Management** - Track stock levels
- **Product Management** - Add/edit products and categories  
- **Order Management** - Process and fulfill orders
- **Vendor Management** - Manage supplier relationships
- **Analytics** - Sales and performance metrics
- **Settings** - Configure payment gateways and system settings
